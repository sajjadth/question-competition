# questio-competition
simple question competition app made with nuxt-js, node-js, mongodb  

### work on backend
```
npm run api
```
### start 
```
npm start
```
### output
```
npm run build
```
create **.env** file in ./api<br/>
and add DB ,PORT, TOKEN_SECRET value to it
